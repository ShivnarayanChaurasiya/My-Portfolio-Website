# 🌐 My Portfolio Website

A modern, responsive, and interactive personal portfolio website built using **HTML, CSS, and JavaScript**. This portfolio showcases my skills, projects, education, and contact information in a clean and user-friendly design.

## 🚀 Live Demo
🔗 https://shivnarayanchaurasiya.github.io/My-Portfolio-Website/

## ✨ Features
- 🎨 Modern and Responsive UI
- 🌙 Dark/Light Mode Toggle
- 👨‍💻 About Me Section
- 💼 Projects Showcase
- 🛠️ Skills Section
- 📄 Resume Download Button
- 📱 Mobile Friendly Design
- 📬 Contact Section
- ⚡ Smooth Animations
- 🔗 Social Media Links
